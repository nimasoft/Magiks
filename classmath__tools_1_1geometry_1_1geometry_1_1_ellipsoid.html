Class Constructor Creates an ellipsoid with given charachteristic matrix A and center.

The equation of the ellipsoid is given as: $$ **(x - c)**^T M **(x - c)** $$

Parameters
MA numpy square matrix representing the charachteristic matrix of the ellipsoid.
centerAn instance of class Point_3D() specifying the ellipsoid center 
For example to have a non-rotated ellipsoid centered at the origin with equation:
x^2/a^2 + y^2/b^2 + z^2/c^2 = 1
Arguments M and center must be selected as:
     [ a^(-2)   0      0     ]
M =  [   0    b^(-2)   0     ]
     [   0      0    c^(-2)  ]

center = (0, 0, 0)
