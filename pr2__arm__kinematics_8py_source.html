753  '''
754  grows the redundant parameter by eta (adds eta to the current phi=q[0] and returns the new value for phi
755  1 - this function does NOT set the configuration so the joint values do not change
756  2 - if the grown phi is not in the feasible interval Delta, it will return the closest point in the range with a safety coefficient k so that
757  new phi = old phi + eta : if eta in Delta
758  new phi = old phi + k*Delta_h : if eta > Delta_h
759  new phi = old phi + k*Delta_l : if eta < Delta_l

769  (dl,dh) = Delta[j]
770  j = j + 1

773  return self.config.q[0] + eta
774  elif eta >= dh: # if zero is not in any of the intervals, this line will raise an error because dh is not defined
