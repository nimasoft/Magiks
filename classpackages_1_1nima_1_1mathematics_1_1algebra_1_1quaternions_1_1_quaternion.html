def __invert__ (   q)

Operator function for the conjugate of a quaternion.

Returns
The conjugate of the given quaternion ( ) 
Conjugate of Quaternion.

>>> q = Quaternion((2, 2, 2, 2))
>>> print(q)
(2 + 2i + 2j + 2k)
>>> print(~q)
(2 - 2i - 2j - 2k)
>>> print(~~q)
(2 + 2i + 2j + 2k)
